# Task 1 - Basic Network Sniffer

## Objective

Build a Python-based network sniffer to capture and analyze network packets.

## Tools Used

* Python 3
* Scapy
* Npcap
* Visual Studio Code

## Features

* Captures live network packets
* Displays Source IP Address
* Displays Destination IP Address
* Identifies Protocols (TCP, UDP, ICMP)
* Shows Port Numbers
* Monitors network traffic in real time

## How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the program:

python Sniffer.py

## Sample Output

Protocol : TCP
Source IP : 192.168.1.33
Destination IP : 203.153.44.44

## Learning Outcomes

* Learned packet sniffing using Scapy
* Understood TCP/IP communication
* Analyzed packet headers and protocols
* Explored network traffic monitoring

## Conclusion

The project successfully captures and analyzes network packets, providing practical experience with network communication and protocol analysis.
